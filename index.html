Fonts

Showcase of all fonts together

This is the header font, Lato.

This is the paragraph font, Merriweather. It is chosen for its higher readability, and as such has serifs to make it easier for the eye to distinguish between characters. This means that even when I, like in this convoluted example, write a very, very long sentence (perhaps even adding a parentheses) we maintain the readability of the paragraph. Isn't that neat?


This is the code font, Fira Code. It is monospaced, making sure all characters take up the same amount of space, which you can tell by looking at how every letter in every line aligns with the ones above.

Here is a sample of how code might look:

    print(f"Hello, number {i}!")

Style Rules

Color choices

  • Use default colors or "black" for all elements.
  • Primarily use main background color. If layering something on top of the background, use the ligther blue default color.
  • Use black for text, except when white or default ligther red is needed to keep it legible.

Aligning and positioning elements

  • Center all elements except longer paragraphs and their subheadings.
  • Use the default container styles.
  • Keep code in a separate box to increase clarity and avoid style clashes.
